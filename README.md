## 📖 About TheHairLocs

**TheHairLocs** is a premium e-commerce website dedicated to providing high-quality natural hair care products specifically designed for locs and textured hair. Our mission is to help individuals master the art of loc care with expert guidance and premium products.

### Tagline
> *"Naturally You. Beautifully Loc'd."*

---

## 🛒 Features

### ✨ Core Features
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Product Catalog** - Browse and shop hair care products by category
- **Shopping Cart** - Side drawer cart with quantity controls
- **Product Filtering** - Filter by category, price range, hair type, and availability
- **Product Ratings & Reviews** - Star ratings display for each product
- **Sale & Discounts** - Current promotion: 20% off on all products
- **Newsletter Subscription** - Stay updated with exclusive offers

### 🛠️ Technical Features
- **Swiper Carousels** - Hero sliders and product carousels
- **Font Awesome Icons** - Consistent iconography throughout
- **WhatsApp Integration** - Direct chat support
- **Back to Top Button** - Easy navigation
- **Multiple Page Types** - Home, Products, Categories, Sale, About, Contact, FAQs, Checkout

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/Hair-Locs.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Hair-Locs
   ```

3. **Open in browser**
   - Simply open `index.html` in your browser
   - Or use a local server like Live Server in VS Code
